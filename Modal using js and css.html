This modal uses both CSS and JS
The title is set to font-size of 16pt.
The background is set to linear-gradient(125deg, #f66, #ff6)
There are two event listeners to open and close the modal. The modal will expand automatically to fit the content it has.
There is a neat padding of 3% all around the body. For the title the padding is 5% on the sides and 2.5% on the top and bottom.
Click anywhere outside the modal to close it!
